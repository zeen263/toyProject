### IoT_mini
2020년 기업 연계 과제로 진행한 IoT 프로젝트입니다. 공개 가능한 범위로 축소해 뼈대만 업로드합니다.<br>
MQTT 프로토콜을 통해 ESP32에서 수집한 센서 데이터를 라즈베리파이에 보냅니다. 라즈베리파이는 해당 토픽을 미리 구독해 뒀다가 메시지를 받으면 웹 페이지에 표시합니다.
+ flask
+ MQTT
